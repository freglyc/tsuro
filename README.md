# Tsuro

Implementation of two to eight player Tsuro for the web. Games are sharable and are played on one or more devices.

View a live version at [tsuro.cfregly.com](https://tsuro.cfregly.com).

[game]: screenshot.png
![alt text][game]

### Run Frontend
```
cd frontend
npm run start
```

### Run Backend
```
cd backend
go run cmd/main.go
```
